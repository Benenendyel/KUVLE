# Learning

## What is this documentation for?

This is just me reflecting on things I learn while building this project.

## What have I learned throughout this project?

I learned a lot throughout this project, specifically the use of CSS frameworks, particularly Tailwind. With Tailwind, I found that building projects with frameworks is much faster and easier, though repetitive at times. I also learned a lot about JavaScript. Coming from Java, where you have to write a lot of code just to make a program do something, JavaScript felt a bit weird at first because there are multiple ways to write the same thing. For example, there are three or more ways to write a function. I am still struggling with callbacks and promises, specifically when and where to use them, though I have a basic understanding of both. I am also still figuring out route handlers and middleware functions. On the other hand, I have gotten comfortable with sending files, and I am making an effort to keep my project organized with proper file structure and naming conventions.
